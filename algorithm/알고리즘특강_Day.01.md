## 1. 기본 입출력

	### (1) 표준 입력 예제 - input()

- 입력 한 줄

- 문자열

- 내장함수 - map(*function*, *iterable*)

  - 정수 입력 1개 받기

    `a = int(input())`

  - 정수 입력 2개 받기

    `b, c = map(int, input().split())`

  - 실수 입력 3개 받기

    `d, e, f = map(float, input(), split())`

  - 문자열 입력 1개

    `n = input()`
