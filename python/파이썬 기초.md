## (2) 문자열

- 이스케이프 코드 
  - `\n` : 문자열 안에서 줄을 바꿀 때 사용
  - `\t` : 문자열 사이에 탭 간격을 줄 때 사용
  - `\\` : 문자 \를 그대로 표현할 때 사용
  -  `\'` : 작은 따옴표를 그대로 표현할 때 사용
  - `\"` : 큰따옴표를 그대로 표현할 때 사용
  - `\r` : 캐리지 리턴(줄 바꿈 문자, 현재 커서를 가장 앞으로 이동)
  - `\f` : 폼 피드(줄 바꿈 문자, 현재 커서를 다음 줄로 이동)
  - `\a` : 벨 소리(출력할 때 PC 스피커에서 '삑'소리가 난다)
  - `\b` : 백 스페이스
  - `\000` : 널 문자
- 문자열 길이 
  - `len()`
- 문자열의 요소는 바꿀 수 없음. 하지만 슬라이싱을 이용해서 바꿀 수 있음
- 문자열 포매팅
  - `%`
    - `%s` - 문자열
    - `%c` - 문자 1개
    - `%d` - 정수
    - `%f` - 부동소수
    - `%o` - 8진수
    - `%x` - 16진수
    - `%%` - Literl %(문자%자체)
  - `"".format()`
  - `f""`
  - 정렬과 공백
    - `"%10s" % "hi"` : hi가 끝에 오게함
  - 소수점 표현하기
    - `"%0.4f" % 3.42134234` : `'3.4213'` -> 0은 자리수, 4는 소수점 뒤에 자르는 숫자
    - 
  - 왼쪽 정렬
    - `"{0:<10}".format("hi")`
  - 오른쪽 정렬
    - `"{0:>10}".format("hi")`
  - 가운데 정렬
    - `"{0:^10}".format("hi")`
  - 공백 채우기
    - `"{0:=^10}".format("hi")`
- 문자열 관련 함수들
  - `count()`
  - 위치 알려주기 
    - `find()` -> 인덱스 위치 출력해줌, 찾는 문자가 없으면 -1 또는 None 값이라고 출력
    - `index()` -> 인덱스 위치 출력해줌, 찾는 문자가 없으면 에러를 출력
  - 문자열 삽입
    - `join()`
  - 소문자를 대문자로 
    - `upper()`
  - 대문자를 소문자로
    - `lower()`
  - 왼쪽 공백 지우기
    - `lstrip()`
  - 오른쪽 공백 지우기
    - `rstrip()`
  - 양쪽 공백 지우기
    - `strip()`
    -

## (5) 딕셔너리 자료형

- 딕셔너리 만들 때 주의 사항
  - 중복 허용 x -> 하나만 처리 
- Key 리스트 만들기
  - `a.keys()` -> key 값 리스트 형태로 출력됨
  - `for k in a.keys(): pritn(k)` -> 키값 줄바꿈 형태로 출력
- values 리스트 만들기
  - `a.values()`
- Key, value 쌍 얻기
  - `a.item()`
- Key, value 쌍 모두 지우기
  - `a.clear()`
- key로 value 얻기
  - `a.get(키값)` -> 없는 키값을 넣으면 None이 출력됨
- `'name' in a` -> a 딕셔너리에 name이 있으면 True 없으며 False

### 3. random

- import random 써줘야함

- 0~1 미만의 실수를 난수로 생성

  - `for x in range(5)`

    `print(random.random())`

- 특정 범위의 실수를 난수로 생성

  - `for x in range(5)`

    `print(random.uniform(1,100))`

- 특정 범위의 정수를 난수로 생성

  - `for x in range(5)`

    `print(random.randint(1,100))`

- 0~1 미만의 실수를 난수로 생성

  - `for x in range(5)`

    `print(random.randrange(1,100,10))`
